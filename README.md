CSE233 – Assignment 2

This repository contains all required materials for Assignment 2 of the Operating Systems course. It includes the code examples from Lab 5, explanations, and all supporting files.

Files Included

process_creation.c – Demonstrates process creation using fork().
file1.c and file2.c – Demonstrate the job of the linker.
simple_program.c – Demonstrates the job of the loader.
describtion.txt – Full explanation of all code examples, the linker, and the loader.
Makefile – Builds all programs with one command.
LICENSE – MIT License.

Build & Run

To compile all programs:
make

To run the programs:
./process_creation
./output_program
./simple_program
